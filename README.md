## Installation

Add this line to your application's Gemfile:

```ruby
gem 'class_diagram', git: 'https://github.com/iiwo/class_diagram', group: 'development'
```

And then execute:

    $ bundle install

## Usage

```ruby
ClassDiagram.diagram(path: 'app/path/to/your/class.rb')
```
